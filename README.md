# PayPal Fee Data

A public, deterministic repository of PayPal merchant fee data generated by the companion [`paypal-fee-crawler`](https://github.com/smeinecke/paypal-fee-crawler).

## Structure

```text
json/          Per-country normalized JSON output
meta/          Country manifest and metadata
schemas/       JSON schemas for the generated files
crawler/       Git submodule pointing to the crawler source
```

## Key files

- `json/{country}.json` — normalized fee tables and derived core fees for a single country
- `json/index.json` — index of all published country files
- `json/core-fees.json` — consolidated core fees across all countries
- `meta/countries.json` — discovered market manifest
- `schemas/paypal-fees-v1.schema.json` — JSON schema for per-country files

## Automated updates

The `.github/workflows/daily-crawl.yml` workflow runs the crawler daily and opens a pull request with any changes.
